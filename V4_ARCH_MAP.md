# Flairy v4.0 Project Charter
- **Role**: High-Performance AI Media Platform (Integrated Monolith)
- **Tech Stack**: Python 3.10+, FastAPI, Jinja2, SQLAlchemy, PostgreSQL, Tailwind CSS
- **Design Philosophy**: 
    1. Zero-Wait I/O (Local file system access via pathlib)
    2. Integrated Architecture (Single server for Web, DB, and Engine)
    3. UI Inheritance (Convert v3 React components to Jinja2 templates)
